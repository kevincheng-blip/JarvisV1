# 📘 第一章｜價格結構 Price Structure（12 指標）

# ==========================================================

這 12 項，是股票短中線最容易預測上漲的結構訊號。

---

## **P01. 趨勢斜率（Trend Slope）**

**定義**:  
用 MA20 / MA60 的斜率判斷趨勢，判斷高點、低點是否遞增（Higher High / Higher Low）。

**核心邏輯**:
- 趨勢向上 = 股票上漲機率最高的基礎
- MA20 斜率 > 0 且高低點結構上升 → 主升段前置訊號

**使用規則**:
- 不得單獨使用，須搭配量能結構
- 趨勢破壞時，所有指標優先降權

---

## **P02. 多頭均線排列（MA20 > MA60 > MA120）**

**定義**:  
三線多頭排列判斷主力成本與趨勢方向。

**核心邏輯**:
- 三線多頭是主升段標準架構
- 股價站上 MA20、MA60 → 主力成本之上

**使用規則**:
- 不適用於極短線盤整股
- 股價 > MA20 > MA60 > MA120 → 最強

---

## **P03. 均線糾結 & 均線突破（MA squeeze breakout）**

**定義**:  
MA 密集糾結表示股價悶盤，一旦突破會噴出。

**核心邏輯**:
- MA 糾結 → 價格壓縮
- 突破 → 動能爆發

**使用規則**:
- 配合成交量確認突破有效性

---

## **P04. K 棒結構（K-line Structure）**

**定義**:  
重點型態識別。

**核心邏輯**:
- **長紅棒**（多方攻擊）：強烈買盤
- **長下影線**（支撐強）：底部支撐
- **吞噬形態**（反轉訊號）：趨勢反轉
- **紅三兵**（強主升）：連續攻擊
- **上影大量**（主力倒貨）：出貨警訊

**使用規則**:
- 需配合成交量與趨勢方向

---

## **P05. 支撐/壓力（Support & Resistance）**

**定義**:  
識別關鍵支撐與壓力位。

**核心邏輯**:
- 越多觸碰點越有效
- 突破壓力 → 容易爆量上漲

**使用規則**:
- 需搭配趨勢與量能判斷

---

## **P06. 缺口（Gap Theory）**

**定義**:  
三種缺口型態。

**核心邏輯**:
- **逃逸缺口** → 最強（主升段中段）
- **突破缺口** → 續漲訊號
- **衰竭缺口** → 高點訊號（準備反轉）

**使用規則**:
- 缺口位置與成交量是關鍵

---

## **P07. 量能結構（Volume Flow）**

**定義**:  
放量上漲 / 縮量回檔的完整比對。

**核心邏輯**:
- 放量上漲（主力進）
- 縮量回檔（洗盤成功）
- 爆量長黑（出貨）

**使用規則**:
- 長黑爆量 = 出貨訊號（權重負向）

---

## **P08. 量價背離（Volume-Price Divergence）**

**定義**:  
價與量的背離關係。

**核心邏輯**:
- 價漲量縮 → 漲不久
- 價跌量縮 → 回檔正常
- 價跌量增 → 崩盤訊號

**使用規則**:
- 背離是趨勢轉弱的先行指標

---

## **P09. 布林帶（Bollinger Band）**

**定義**:  
布林帶位置與寬度。

**核心邏輯**:
- 突破上軌＝動能噴發
- 下軌縮口→即將爆發

**使用規則**:
- 需配合其他技術指標確認

---

## **P10. RSI / KD / MACD（Momentum 指標）**

**定義**:  
動能指標組合。

**核心邏輯**:
- **RSI 50** 是多空分界
- **KD <20 金叉** = 買點
- **MACD 金叉** = 趨勢翻多

**使用規則**:
- 三個指標同時看多 → 動能最強

---

## **P11. ATR 波動度（Volatility）**

**定義**:  
平均真實波幅（Average True Range）。

**核心邏輯**:
- 波動高 → 風險大
- 波動低 → 下一波段前的靜止

**使用規則**:
- 低波動 + 突破 → 後續動能強

---

## **P12. 套牢壓力（VAP Distribution）**

**定義**:  
檢查上方成交密集區是否形成壓力。

**核心邏輯**:
- VAP 無壓力 → 容易噴
- 套牢越深 → 越難漲

**使用規則**:
- 需搭配趨勢、量能同時判斷
- 股價突破壓力區後，VAP 向上翻轉最為關鍵

---

# 📘 第二章｜籌碼 Capital Flow（9 指標）

# ==========================================================

**台股「80% 的漲跌」來自籌碼。**

---

## **C01. 外資買賣超（Foreign Investors）**

**定義**:  
外資連續買超天數與買超金額。

**核心邏輯**:
- 外資連買 ≥3 天 → 中線漲
- 外資連賣 → 停看聽

**使用規則**:
- 外資與投信「同步偏多」＝最強的上漲預兆

---

## **C02. 投信買賣超（Investment Trust）**

**定義**:  
投信連續買超天數與買超金額。

**核心邏輯**:
- **台股最準！**  
- 投信連買 ≥3 天 → 超強訊號

**使用規則**:
- 投信連續買超比外資更可靠

---

## **C03. 自營商（Dealer）**

**定義**:  
自營商買賣超。

**核心邏輯**:
- 偏避險，短線參考性為主

**使用規則**:
- 自營商買超通常偏短線

---

## **C04. 大戶持股比（Major Holders Ratio）**

**定義**:  
前 15% 大戶持股變化。

**核心邏輯**:
- 大戶持續加碼 → 主力吸籌
- 大戶減 → 主力出貨

**使用規則**:
- 散戶比率需同步降低才有效
- 大戶比率上升 + 散戶下滑 → 股價最容易漲

---

## **C05. 散戶比率（Retail Ratio）**

**定義**:  
散戶持股比例佔比。

**核心邏輯**:
- 散戶越多 → 越容易跌  
- 散戶越少 → 主力越容易拉

**使用規則**:
- 極端時（散戶大增）權重負向
- 散戶比率低於均值 → 是安全水位

---

## **C06. 分點籌碼（Broker Branch）**

**定義**:  
特定券商分點買賣超。

**核心邏輯**:
- 凱基松山、摩根大通、元大總公司  
- 連續買一定有事

**使用規則**:
- 需追蹤特定主力分點動向

---

## **C07. 主力成本（主力加權平均）**

**定義**:  
主力持倉的加權平均成本。

**核心邏輯**:
- 股價 > 主力成本 → 多頭勝利
- 股價 < 主力成本 → 主力倒貨

**使用規則**:
- 主力成本是關鍵支撐/壓力

---

## **C08. 融資（Margin）**

**定義**:  
融資餘額變化。

**核心邏輯**:
- 融資爆量 → 散戶高風險

**使用規則**:
- 融資大增 → 籌碼不穩

---

## **C09. 融券（Shorting）**

**定義**:  
融券餘額變化。

**核心邏輯**:
- 融券大增 → 有機會軋空

**使用規則**:
- 融券大增 + 股價上漲 → 軋空行情

---

# 📘 第三章｜財報 Fundamental（8 指標）

# ==========================================================

**股票長期上漲靠財報。**

---

## **F01. 營收 YOY / MOY**

**定義**:  
營收年增率與月增率。

**核心邏輯**:
- 連續三季年增 → 長線大多頭

**使用規則**:
- 單季不可單獨使用，需看 3 季趨勢
- 三率三升 = 中長線主升段

---

## **F02. 毛利率（Gross Margin）**

**定義**:  
毛利率趨勢。

**核心邏輯**:
- 毛利連 3 季上升 → 超強信號  
- 毛利下滑 → 基本面惡化

**使用規則**:
- 高毛利股比低毛利股權重更高
- 毛利率提升＝產品競爭力增加

---

## **F03. 營益率（Operating Margin）**

**定義**:  
營業利益率。

**核心邏輯**:
- 代表管理效率是否提升

**使用規則**:
- 營益率上升 → 營運效率改善

---

## **F04. EPS（每股盈餘）**

**定義**:  
每股盈餘趨勢。

**核心邏輯**:
- 持續穩定上升 → 長線標的

**使用規則**:
- EPS 年增率 > 20% → 成長股

---

## **F05. 自由現金流 FCF**

**定義**:  
營運現金流 − 資本支出。

**核心邏輯**:
- FCF > 0 = 真成長  
- FCF < 0 = 燒錢公司

**使用規則**:
- 3 年負 FCF → 長期降評
- 正 FCF + 成長 → 中長期股價最穩定的公司

---

## **F06. ROE / ROA**

**定義**:  
股東權益報酬率與資產報酬率。

**核心邏輯**:
- ROE 是股東報酬最重要指標

**使用規則**:
- ROE > 15% → 優質公司
- ROA > 10% → 資產運用效率高

---

## **F07. 負債比（Liabilities Ratio）**

**定義**:  
負債比率。

**核心邏輯**:
- 輕資產高毛利公司通常較強

**使用規則**:
- 負債比過高 → 財務風險

---

## **F08. 股東權益成長（Equity Growth）**

**定義**:  
股東權益變化。

**核心邏輯**:
- 股東權益增加＝公司體質變強

**使用規則**:
- 股東權益持續成長 → 長線多頭

---

# 📘 第四章｜題材 Catalyst（7 指標）

# ==========================================================

**股票是「新聞推動的動物」。**

---

## **K01. 法說會（Guidance）**

**定義**:  
法說會財測指引。

**核心邏輯**:
- 財測上修 → 大漲  
- 財測下修 → 直接跌停

**使用規則**:
- 法說會上修 + 籌碼強 = 大波段

---

## **K02. 新訂單（AI / 車用 / 伺服器）**

**定義**:  
重大新訂單公告。

**核心邏輯**:
- 新訂單是公司最強的基本面催化劑

**使用規則**:
- AI / 車用 / 伺服器訂單 → 近年最強題材

---

## **K03. 擴產 / 併購（M&A）**

**定義**:  
擴產計劃或併購案。

**核心邏輯**:
- 半導體最常見的利多來源

**使用規則**:
- 擴產 → 未來成長動能

---

## **K04. 接入大客戶（Apple / Nvidia / TSMC）**

**定義**:  
成為大廠供應鏈。

**核心邏輯**:
- 供應鏈題材最容易噴

**使用規則**:
- 接入 Apple / Nvidia / TSMC → 股價爆發

---

## **K05. 政策題材（綠能 / 軍工）**

**定義**:  
政策支持的產業題材。

**核心邏輯**:
- 政策保護行情 → 走長線

**使用規則**:
- 綠能 / 軍工 / 生技 → 政策保護

---

## **K06. ETF 變化（被納入 / 剔除）**

**定義**:  
被納入或剔除重要指數/ETF。

**核心邏輯**:
- MSCI / 富時 / ETF 變動  
- 近年最影響股價的因素之一

**使用規則**:
- 被納入 → 被動資金買盤
- 被剔除 → 被動資金賣壓

---

## **K07. 熱門題材（AI / 生技 / 軍工）**

**定義**:  
當前市場熱門題材。

**核心邏輯**:
- 題材王永遠是盤面最強

**使用規則**:
- AI / 生技 / 軍工 → 輪流成為主流

# 📘 第六章｜量化風控 Quant（6 指標）

# ==========================================================

**主力與外資必看。**

---

## **Q01. Sharpe Ratio**

**定義**:  
風險調整後報酬率。

**核心邏輯**:
- 檢驗可否承擔這檔股票的風險

**使用規則**:
- Sharpe > 2.0 → 優質標的

---

## **Q02. 波動度（Volatility / ATR）**

**定義**:  
股價波動度。

**核心邏輯**:
- 波動越低 → 越適合中長線

**使用規則**:
- 低波動 + 高報酬 → 最佳標的

---

## **Q03. 最大回撤 MDD**

**定義**:  
Maximum Drawdown。

**核心邏輯**:
- 長期投資必看  
- MDD 高 → 心臟要強

**使用規則**:
- MDD < 15% → 風險可控

---

## **Q04. Beta（與大盤之連動）**

**定義**:  
與大盤的相關性係數。

**核心邏輯**:
- Beta 高 → 上漲時漲更多，崩時跌更多

**使用規則**:
- Beta > 1.0 → 高彈性
- Beta < 1.0 → 防禦性

---

## **Q05. 因子暴露（Momentum / Value / Quality）**

**定義**:  
多因子模型的因子暴露度。

**核心邏輯**:
- 掌握該股真正的性格

**使用規則**:
- Momentum 高 → 動能股
- Value 高 → 價值股
- Quality 高 → 優質股

---

## **Q06. 集中度（Portfolio Weight）**

**定義**:  
投資組合中的集中度。

**核心邏輯**:
- 單檔過度集中 → 風險爆棚

**使用規則**:
- 單檔不超過 20% → 風險控制

## 評分標準

| 滿足指標數 | 評級 | 說明 |
|-----------|------|------|
| **≥ 8** | 主升段候選 | 高概率上漲，適合積極參與 |
| **5～7** | 強勢股 | 有上漲機會，需持續觀察 |
| **≤ 3** | 無上漲空間 | 不建議進場，等待更好時機 |
| **籌碼逆轉** | 立即警戒 | 散戶湧入 / 大戶減碼 → 準備出場 |

## 使用流程

1. **收集 60 指標數據**
2. **計算各指標滿足程度**
3. **加權計算總分**
4. **判斷 Verdict（STRONG_BUY / BUY / NEUTRAL / AVOID）**
5. **結合 Path B/C 驗證結果**
6. **Path D RL 優化參數**

---

## 與 J-GOD 系統的整合

### Path A（Backtest Engine）
- 使用這 60 指標作為 Alpha Engine 的因子來源
- 計算各指標值並整合到 Alpha 訊號

### Path B（Walk-Forward Analysis）
- 在 Walk-Forward 測試中驗證 60 指標的穩定性
- 觀察哪些指標在不同市場環境下最有效

### Path C（Scenario Lab）
- 使用不同指標組合建立 Scenario
- 比較不同指標權重的策略表現

### Path D（RL Engine）
- 60 指標作為 State Space 的一部分
- Reward Function 考慮指標滿足程度

### Path E（Live Trading）
- 即時計算 60 指標值
- 根據指標組合生成交易決策
