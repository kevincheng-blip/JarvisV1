# J-GOD 知識庫結構化處理說明

本目錄包含 14 本 AI 知識庫文件的結構化版本，每本文件會產生三個版本：

## 版本說明

1. **`*_STRUCTURED.md`**：忠於原文，只整理
   - 建立清楚的目錄與分級標題
   - 將概念、規則、算式、案例、操作步驟分區整理
   - 內容語句盡量忠於原文
   - 不做任何「糾錯」，只整理

2. **`*_ENHANCED.md`**：說清楚、拆邏輯
   - 在 STRUCTURED 基礎上
   - 把跳躍的推導拆成清楚步驟
   - 增加白話解釋與實作角度說明
   - 標記 [疑義／待確認] 但不多做修正

3. **`*_CORRECTED.md`**：允許上網查、補強、糾錯
   - 在 ENHANCED 基礎上
   - 補齊缺的公式、統計定義、技術指標標準寫法
   - 主動修正疑似錯誤或不完整的邏輯
   - 標記來源：[原文]、[重寫整理]、[補充說明]、[外部知識補強]、[疑義／待確認]、[修正建議]

## 標記系統

- **[原文]**：來自原書
- **[重寫整理]**：重構語句、拆成步驟、換說法，但邏輯相同
- **[補充說明]**：多解釋的地方
- **[外部知識補強]**：根據訓練知識 + 網路資料補上的內容
- **[疑義／待確認]**：可能錯、或多種版本，需要之後決定
- **[修正建議]**：修正版本與說明
